# JavaScript Array Exercises

## Description

This project contains a collection of JavaScript exercises focused on array manipulation, iteration techniques, callbacks, and higher-order functions.

The exercises cover essential JavaScript concepts such as loops, array methods, multidimensional arrays, filtering, searching, aggregation, and the use of modern features like the Spread Operator.

## Features

### Array Manipulation

* Concatenate multiple arrays.
* Create subarrays using `slice()`.
* Modify arrays using `splice()`.

### Iteration Techniques

* Iterate through arrays using:

  * `for`
  * `for...of`
  * `forEach()`

### Mathematical Operations

* Calculate sums using loops and `reduce()`.
* Calculate averages of array elements.
* Find maximum and minimum values.

### Search Operations

* Find specific values in arrays.
* Retrieve element positions.
* Search for items using `find()` and `findIndex()`.

### Filtering Data

* Filter even numbers.
* Filter multiples of 3 greater than 5.
* Create custom filtering functions.

### Higher-Order Functions

* Implement callback functions.
* Apply operations to entire arrays.
* Use `map()` for data transformation.

### Spread Operator

* Merge multiple arrays using the Spread Operator (`...`).

### Multidimensional Arrays

* Create and access complex 3D array structures.
* Retrieve specific elements from nested arrays.

### Data Deduplication

* Remove duplicate values using `Set`.
